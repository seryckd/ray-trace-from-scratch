# ray-trace-from-scratch
Building the Ray Tracer from the book 'Computer Graphics from Scratch'

[Try it out](https://seryckd.github.io/ray-trace-from-scratch/)

Features
* Geometry
  * Spheres
* Light
  * Sources - Ambient, Point, Directional
  * Specular Reflection
  * Shadows
  * Reflections
* Optimizations
